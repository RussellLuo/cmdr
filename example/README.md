Example
=======

A simple example that shows how to use `Cmdr`.


Usage
-----

1. Install `Cmdr`

        $ cd /tmp
        $ git clone https://github.com/RussellLuo/cmdr.git
        $ cd cmdr
        $ python setup.py install

2. Set the environment variable

        $ export CMDR_CONFIG_YAML=/tmp/cmdr/example/cmdr.yml

3. Run commands

        $ cmdr hello
        $ cmdr goodbye
